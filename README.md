# GridImgPDF

> Turn any images into a grid PDF — free, offline, no upload needed.

**🌐 Live:** [monirwpc.github.io/gridimgpdf](https://monirwpc.github.io/gridimgpdf/)

---

## Overview

**GridImgPDF** is a lightweight, browser-based tool that lets you combine multiple images into a neat grid layout and export them as a print-ready A4 PDF — all without installing any software or uploading files to a server.

Everything runs locally in your browser. Your images never leave your device.

---

## Features

- 📁 **Drag & drop or browse** — select any number of JPG, PNG, or WEBP images
- 🔢 **Multiple grid layouts** — choose from 2×2, 3×2, 2×3, 3×3, 4×2, or 4×3
- 📐 **Adjustable margin & gap** — fine-tune spacing with live sliders
- 📄 **A4 output** — print-ready PDF with proper dimensions
- ⚡ **Instant download** — PDF is generated and downloaded in seconds
- 🔒 **100% private** — no server, no upload, no tracking
- 📴 **Works offline** — once loaded, no internet required

---

## How to Use

1. Open the [live tool](https://monirwpc.github.io/gridimgpdf/)
2. Click **"Click to browse"** or drag & drop your screenshots
3. Choose a **grid layout** (e.g. 3×2 for 6 screenshots per page)
4. Adjust **margin** and **gap** using the sliders if needed
5. Click **"Generate PDF"**
6. Click **"Download PDF"** — done!

## Tech Stack

- **HTML5** — structure
- **CSS3** — styling and layout
- **Vanilla JavaScript** — logic and file handling
- **[jsPDF](https://github.com/parallax/jsPDF)** — client-side PDF generation
- **[Tabler Icons](https://tabler-icons.io/)** — icon set
- **[Google Fonts](https://fonts.google.com/)** — Syne + DM Mono typography
- **GitHub Pages** — hosting

No frameworks. No build tools. Just a single HTML file.
